How It Works

Meta-Tool Pattern

  • Instead of exposing 68 individual MCP tools, WP MCP Ultimate exposes only 3 meta-tools
  • This keeps the AI's tool list clean and prevents context window bloat
  • The AI discovers abilities dynamically, then executes them through execute-ability
  • New abilities (from other plugins) are auto-discovered — no config changes needed

Replaces 3 Plugins

  • abilities-api — WordPress Abilities API polyfill (bundled)
  • mcp-adapter — MCP protocol server (bundled)
  • mcp-expose-abilities — 57 WordPress abilities (bundled + extended)
  • One plugin, one endpoint, zero dependencies

Security Model

  • WordPress Application Password auth over HTTPS
  • MCP session tracking via Mcp-Session-Id header
  • Each ability checks WordPress capabilities (e.g., edit_posts)
  • Double permission check: MCP layer + WordPress capability system
Setup in 4 Steps
Get WP MCP Ultimate working with Claude Code in under 5 minutes.
1

Install Plugin

Upload the plugin zip to WordPress via Plugins → Add New → Upload, or use WP-CLI:

wp plugin install wp-mcp-ultimate.zip --activate

Deactivate the old plugins if you had them: abilities-api, mcp-adapter, mcp-expose-abilities.

2

Create Application Password

In WordPress admin: Users → Your Profile → Application Passwords

Give it a name (e.g., "Claude Code") and click Add New. Copy the password.

The user must have Administrator role for full access to all abilities.

3

Configure Claude Code MCP

Add to ~/.claude/settings.json under mcpServers:

"wordpress": { "command": "npx", "args": ["-y", "@automattic/mcp-wordpress-remote@latest"], "env": { "WP_API_URL": "https://yoursite.com/wp-json/mcp/wp-mcp-ultimate", "WP_API_USERNAME": "your_username", "WP_API_PASSWORD": "xxxx xxxx xxxx xxxx xxxx xxxx" } }
4

Use It!

The MCP adapter exposes 3 tools in Claude Code:

wp-mcp-ultimate-discover-abilities — list all abilities

wp-mcp-ultimate-get-ability-info — get ability schema

wp-mcp-ultimate-execute-ability — run any ability

Just ask Claude: "List my WordPress posts" or "Create a draft post about X"

Tool Name Issue
Current compatibility note with @automattic/mcp-wordpress-remote

Known Issue

  • The Automattic MCP proxy expects tool names prefixed with mcp-adapter- (matching the old MCP Adapter plugin)
  • WP MCP Ultimate registers tools as wp-mcp-ultimate-* instead
  • The proxy's discover-abilities call fails because it looks for mcp-adapter-discover-abilities
  • Workaround: Use curl or direct JSON-RPC calls to the endpoint, or rename the tool prefix in the plugin's DefaultServerFactory.php
  • Alternative: The plugin's admin dashboard (WP Admin → MCP Ultimate) can generate a Claude Code config with the correct endpoint

Direct JSON-RPC Examples
Use these with curl or any MCP-compatible client. Remember to initialize a session first.

Initialize Session

# Step 1: Initialize and capture session ID curl -s -D - \ -u "USERNAME:APP_PASSWORD" \ "https://yoursite.com/wp-json/mcp/wp-mcp-ultimate" \ -H "Content-Type: application/json" \ -d '{ "jsonrpc": "2.0", "method": "initialize", "id": 1, "params": { "protocolVersion": "2025-03-26", "capabilities": {}, "clientInfo": { "name": "my-client", "version": "1.0" } } }' # Look for: Mcp-Session-Id: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

List Recent Posts

# Use session ID from initialize curl -s \ -u "USERNAME:APP_PASSWORD" \ "https://yoursite.com/wp-json/mcp/wp-mcp-ultimate" \ -H "Content-Type: application/json" \ -H "Mcp-Session-Id: YOUR_SESSION_ID" \ -d '{ "jsonrpc": "2.0", "method": "tools/call", "id": 2, "params": { "name": "wp-mcp-ultimate-execute-ability", "arguments": { "ability_name": "content/list-posts", "parameters": { "per_page": 5, "status": "publish" } } } }'

Create a Draft Post

curl -s \ -u "USERNAME:APP_PASSWORD" \ "https://yoursite.com/wp-json/mcp/wp-mcp-ultimate" \ -H "Content-Type: application/json" \ -H "Mcp-Session-Id: YOUR_SESSION_ID" \ -d '{ "jsonrpc": "2.0", "method": "tools/call", "id": 3, "params": { "name": "wp-mcp-ultimate-execute-ability", "arguments": { "ability_name": "content/create-post", "parameters": { "title": "My AI-Generated Post", "content": "<p>Hello from MCP!</p>", "status": "draft" } } } }'

Search Content

curl -s \ -u "USERNAME:APP_PASSWORD" \ "https://yoursite.com/wp-json/mcp/wp-mcp-ultimate" \ -H "Content-Type: application/json" \ -H "Mcp-Session-Id: YOUR_SESSION_ID" \ -d '{ "jsonrpc": "2.0", "method": "tools/call", "id": 4, "params": { "name": "wp-mcp-ultimate-execute-ability", "arguments": { "ability_name": "content/search", "parameters": { "query": "collagen", "type": "post", "per_page": 10 } } } }'
